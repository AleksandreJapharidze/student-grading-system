## To run the server, clone the repository and execute the following commands (assuming you already have node and npm set up on your machine):
### 1. cd server
### 2. npm install
### 3. tsc
### 4. npm start

## To run the tests, clone the repository the same way as above, but execute the following commands:
### 1. cd server
### 2. npm install
### 3. tsc
### 4. npm test
